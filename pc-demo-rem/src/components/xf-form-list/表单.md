# 配置 xf-form-list

## 参数

- label form-item的 label
- key 属性名称
- listeners 事件
- attrs 组件 attrs
- component 自定义组件名称

# 拓展---自定义组件

```
     component: '自定义组件名称',
     import HelloWorld from '@components/hello-world/hello-world.vue'
     // {
     //   id: 12,
```
