# 配置 xf-table-list

## 参数

- showPage 展示页面
- pages {
  attr: {},
  pageIndex: 1, // 页数
  pageSize: 10, // 条数
  total: 0 // 总条数
  }
- isEdit 是否编辑
- data 数据 array
- columns 表格配置 按照 el-table-column attr配置 注意 slot 插槽

# 拓展---自定义组件 插槽 slot

```
let columns = ref([
  {
    label: '身高',
    prop: 'value2',
    align: 'center',
    headerAlign: 'center',
    width: 200,
    slot: 'value2Slot' // 插槽
  }
]
<!--  插槽        -->
  <template #value2Slot="{scope}"> // 匹配对应
      <el-button type="primary">
        {{ scope.row[scope.item.prop] }}
       </el-button>
  </template>

```
